    Find us on GitHub

    All of the Genesis code lives on Github, because we're all about Open Source. We use issues to track problems, pull requests for code changes, and releases for, well, releases.


    Genesis itself contains the source code for the genesis utility.
    https://github.com/genesis-community/genesis

    Related projects live under the genesis-community Github org:
    https://github.com/genesis-community

    Join Our Call

    We're always happy for people to get involved with Genesis and help out. We hold a monthly Genesis Roadmap Call where you can get progress updates and participate in the conversation!

    Work with us on Trello

    We coordinate a loose-knit group of hackers on the Internet via our public Trello Board. Whether you want to get involved and pick up some cards, or just keep pace with the state of Genesis, check it out!
