Genesis Installation
Installing Genesis is quick and painless on any system.



On Ubuntu/Debian you can use the following commands:

wget -q -O - https://raw.githubusercontent.com/starkandwayne/homebrew-cf/master/public.key | apt-key add -
echo "deb http://apt.starkandwayne.com stable main" | tee /etc/apt/sources.list.d/starkandwayne.list
apt-get update
apt-get install genesis -y


On OS X/Mac use the following commands:


brew tap starkandwayne/cf
brew tap cloudfoundry/tap
brew install genesis spruce safe bosh-cli vault git

On Centos 7 / Amazon Linux use these commands


Install Dependencies:

sudo yum install -y perl perl-Data-Dumper perl-Time-Local perl-Time-Piece \
  perl-local-lib perl-Carp perl-PathTools perl-Digest perl-File-Temp perl-Socket

Grab latest url from releases page:


wget https://github.com/genesis-community/genesis/releases/latest
chmod 0755 genesis

genesis requires Perl, Perl is everywhere so this shouldn't be a problem.


You will also need to set up Git:


git config --global user.name "Your Name"
git config --global user.email your@email.com
